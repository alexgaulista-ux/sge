Bem-vindo de Volta!

Nota Importante: Este sistema agora utiliza Cloudflare Workers e D1 para persistência de dados e gerenciamento de usuários.
Para acessar: Você precisará de um backend Cloudflare Worker configurado para se conectar ao seu D1. As credenciais de login dependem dos usuários que você cadastrar neste backend.
Este é apenas o frontend. A implementação do Cloudflare Worker e do D1 não está incluída neste pacote, pois são componentes de backend. Você precisará configurá-los separadamente.
